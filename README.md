# School Management System (React + Node/Express + MongoDB)

A modern school management system with roles for **Admin**, **Teacher**, and **Parent**.  
Admins can assign teachers to subjects, register new students or teachers, and manage communication.  
Teachers can log in to enter exam results, view assigned classes, and message parents.  
Parents can log in to view student performance and progress.

---

## Features

### ✅ Admin
- Login to secure Admin Dashboard
- Assign teachers to subjects per class
- Prevent duplicate assignments
- View current assignments
- Register new students and teachers
- Manage alerts/messages to students or parents

### ✅ Teacher
- Login and view dashboard
- Enter student exam scores per class & subject
- View classes assigned to them
- Send messages to students/parents

### ✅ Parent
- Login with account
- View student results and messages

---

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React + Bootstrap |
| Backend | Node.js + Express |
| Database | MongoDB (Mongoose) |
| Auth | JWT + LocalStorage persistence |
